交互示例：代码翻译

这是核心教学元素之一。左侧是真实代码，右侧是通俗易懂的中文。

代码 
async function fetchUserData(userId) {
  const response = await fetch(`/api/users/${userId}`);
  const data = await response.json();
  return data;
}
通俗易懂的中文

创建一个获取用户信息的函数...

向服务器请求这个特定用户的数据...

将服务器的响应从JSON格式转换为JavaScript...

返回我们收到的用户数据。

💡 顿悟时刻

await 关键字就像说"在这里暂停，直到我们得到答案" - 它让你的代码保持简洁易读，而不是使用回调金字塔。
