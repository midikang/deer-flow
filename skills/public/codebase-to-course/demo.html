Transform any codebase into a beautiful, interactive HTML course

English 中文

What This Skill Does

The codebase-to-course skill converts any repository into an interactive single-page HTML course with scroll-based modules, animated visualizations, quizzes, and plain-English code explanations.

📚 Scroll-Based Modules

Smooth navigation through 5-8 learning modules, each teaching a specific aspect of the codebase.

🎨 Animated Visualizations

Data flow diagrams, component interactions, and architecture animations that bring code to life.

✅ Interactive Quizzes

Test practical application with scenario-based questions, not just memory recall.

📝 Code Translations

Side-by-side code and plain English explanations, making complex code accessible.

💡 Glossary Tooltips

Every technical term explained on hover - no prior knowledge assumed.

🎯 Group Chat Animations

iMessage-style conversations between components showing how they communicate.

Interactive Example: Code Translation

This is one of the core teaching elements. Real code on the left, plain English on the right.

CODE 
async function fetchUserData(userId) {
  const response = await fetch(`/api/users/${userId}`);
  const data = await response.json();
  return data;
}
PLAIN ENGLISH

Create a function that gets user information...

Ask the server for data about this specific user...

Convert the server's response from JSON format into JavaScript...

Send back the user data we received.

💡 Aha! Moment

The await keyword is like saying "pause here until we get an answer" - it keeps your code clean and readable instead of using callback pyramids.

How to Use This Skill

When interacting with DeerFlow, trigger this skill by saying:

  • 💬 "Turn this into a course"
  • 💬 "Make a course from this project"
  • 💬 "Create an interactive tutorial from this codebase"
  • 💬 "Turn https://github.com/user/repo into a course"
  • 💬 "Make a course from ./my-project"

Design Philosophy

Courses are designed for "vibe coders" - people who build software using AI tools but want to understand what's happening under the hood.

🎯 Learning Goals

• Steer AI coding tools more effectively
• Detect when AI is wrong or hallucinating
• Debug issues and break out of bug loops
• Make smarter architectural decisions
• Acquire precise technical vocabulary
