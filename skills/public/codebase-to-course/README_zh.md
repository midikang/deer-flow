# 代码库转课程技能

[English](./README.md) | 中文

## 概述

**代码库转课程（codebase-to-course）** 技能可以将任何代码库转换为精美的交互式单页HTML课程，向非技术人员（特别是使用AI构建软件的"氛围程序员"）讲解代码的工作原理。

## 功能特性

将代码仓库转换为独立的HTML文件，具有以下特性：
- 📚 **基于滚动的模块** 流畅导航
- 🎨 **动画可视化** 展示数据流和架构
- ✅ **交互式测验** 测试实际应用
- 📝 **代码↔中文翻译** 并排显示通俗易懂的解释
- 💡 **术语表提示** 涵盖每个技术术语
- 🎯 **群聊动画** 展示组件交互
- 📊 **流程图** 说明系统架构

## 使用方法

在与DeerFlow交互时，您可以通过以下方式触发此技能：

- "将此代码库转换为课程"
- "从这个项目制作课程"
- "从这个代码库创建交互式教程"
- "交互式讲解这个代码库"
- "将 [GitHub URL] 转换为课程"
- "从 ./my-project 制作课程"

## 使用示例

### 转换当前代码仓库：

将DeerFlow代码仓库转换为交互式课程
```

```

```

```

## 生成内容

该技能创建一个 `course.html` 文件，包含：

1. **5-8个模块** 构建学习旅程
2. **模块1**：应用的功能和工作原理
3. **模块2-6**：组件、数据流、API、模式、调试
4. **最终模块**：整体架构全景

每个模块包括：
- 3-6个屏幕（子部分）
- 代码及其中文翻译
- 交互元素（测验、动画）
- "顿悟！"提示框
- 实用比喻

## 设计理念

该课程专为 **"氛围程序员"** 设计，他们：
- 使用AI编码工具构建软件
- 需要理解代码而不必从头编写
- 希望更好地引导AI、更快调试、做出更明智的决策
- 可能缺乏传统的计算机科学教育

每个概念都用通俗易懂的语言解释，假设学习者零技术背景。

## 文件结构

```
skills/public/codebase-to-course/
├── SKILL.md                           # 主要技能实现指南
├── README.md                          # 英文文档
├── README_zh.md                       # 本文件（中文文档）
└── references/
    ├── design-system.md              # CSS设计令牌、颜色、排版
    └── interactive-elements.md       # 交互元素模式
```

## 技术细节

- **输出**：单个独立HTML文件
- **依赖**：仅Google Fonts CDN
- **大小**：通常200-500KB
- **离线工作**：是（初次字体加载后）
- **移动友好**：完全响应式
- **浏览器支持**：现代浏览器（Chrome、Firefox、Safari、Edge）

## 实现流程

该技能遵循4个阶段的流程：

1. **代码库分析** - 深入了解代码仓库结构
2. **课程设计** - 规划5-8个模块的学习路径
3. **构建课程** - 生成包含所有交互元素的HTML
4. **审查和打开** - 向用户展示以获取反馈
